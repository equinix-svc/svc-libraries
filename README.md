# Installing SVC Python Libraries

### Install NetBox lib
```
pip install "git+ssh://git@github.com/<org>/svc-libraries.git#subdirectory=packages/svc_netbox_lib"
```

### Install Juniper lib
```
pip install "git+ssh://git@github.com/<org>/svc-libraries.git#subdirectory=packages/svc_juniper_lib"
```

### Install Synchronize lib
```
pip install "git+ssh://git@github.com/<org>/svc-libraries.git#subdirectory=packages/svc_synchronize_lib"
```

### Install with version tag
```
pip install "git+ssh://git@github.com/<org>/svc-libraries.git@v0.1.0#subdirectory=packages/svc_netbox_lib"
```
